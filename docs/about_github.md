## 解决访问时图裂、加载慢的问题。

> 替换hosts文件

项目地址：https://github.com.cnpmjs.org/521xueweihan/GitHub520

```bash
# Linux编辑hosts
sudo nano /etc/hosts

# Window编辑hosts
C:\Windows\System32\drivers\etc\hosts
```

## 使用CDN节点

例子：https://cdn.jsdelivr.net/gh/shaonianzhentan/ha-docs@master/README.md
